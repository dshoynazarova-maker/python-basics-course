# Python Basics — Full Course Repository

A 16-week beginner Python course covering core programming concepts —
from variables and functions to APIs, file handling, and Pythonic coding.

---

| Week | Topic | Type |
|------|-------|------|
| Week 1  | Course Introduction and First Concepts              | Lecture  |
| Week 2  | Variables, Data Types and Operators                 | Lecture  |
| Week 3  | Selection Control & Repetition                      | Lecture  |
| Week 4  | Functions                                           | Lecture  |
| Week 5  | Lists                                               | Lecture  |
| Week 6  | Advanced Lists, Tuples & Linear Search              | Lecture  |
| Week 7  | Revision Week                                       | Revision |
| Week 8  | Midterm Exam                                        | Exam     |
| Week 9  | Advanced String Processing                          | Lecture  |
| Week 10 | Dictionaries                                        | Lecture  |

| Week 12 | File I/O                                            | Lecture  |
| Week 13 | Modules and Interacting with the Web (APIs)         | Lecture  |
| Week 14 | Intermediate Data Structures & Pythonic Coding      | Lecture  |
| Week 15 | Revision Week                                       | Revision |
| Week 16 | Final Exam                                          | Exam     |

---

> Weeks 8 and 16 are exams — no code folders needed.

---

## How to run
```bash
cd week1/ex1
python main.py
```

Python 3.10+ recommended. No external libraries needed for most weeks.  
Week 13 (APIs) requires the `requests` library: `pip install requests`
